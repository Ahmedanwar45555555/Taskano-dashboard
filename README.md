# Taskano Dashboard

Taskano Dashboard is a simple study productivity dashboard built using HTML, CSS, and JavaScript.

## Features

* Task Management

  * Add and delete study tasks.
* Timeline Planner

  * Create events with dates and track upcoming deadlines.
* Notes Section

  * Save quick study notes.
* Pomodoro Timer

  * 25-minute focus timer to improve productivity.
* Habit Tracker

  * Track daily study habits using checkboxes.
* Search & Filter

  * Search through dashboard sections quickly.

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Project Structure

```text
index.html
style.css
study.js
```

## Learning Goals

This project was created to practice:

* DOM Manipulation
* Event Handling
* Dynamic Content Creation
* CSS Styling and Layouts
* JavaScript Timer Logic
* Search and Filtering Features
